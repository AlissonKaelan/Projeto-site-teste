Texto HTML

Um texto HTML é um arquivo escrito na linguagem de marcação HTML (HyperText Markup Language), usado para estruturar o conteúdo de uma página web. Ele é composto por tags que definem elementos como títulos, parágrafos, imagens, links, tabelas e muito mais.

Explicação:

Declaração do DOCTYPE: indica que o documento está em HTML5. Elemento : É o contêiner principal de toda a página. O atributo lang="pt-BR" define o idioma como português do Brasil. Cabeçalho (): Contém informações como o título da página e configurações de codificação (UTF-8 para caracteres especiais). Corpo (): Abriga o conteúdo visível da página, como seções, parágrafos e links.
